# nestjs-backend

NestJS 기반의 백엔드 라이브 REST API 서비스 입니다.
타 프로젝트의 백엔드 기반이 됩니다.

## Stack

```bash
NestJS
Typescript
Supabase - DB
Render - HOSTING
```

## 주요 패키지

```bash
class-validator - 데이터 유효성 검사
jest - 유닛테스트
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
